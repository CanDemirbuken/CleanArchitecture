CleanArchitecture is a hands-on ASP.NET Core 10 project built to learn and apply modern software architecture principles in a production-oriented manner.

The project follows the Clean Architecture approach by separating responsibilities into independent layers while demonstrating common backend development patterns and best practices used in enterprise applications.

---

## 🚀 Technologies

- ASP.NET Core 10

- Clean Architecture
- CQRS

## 🎯 Purpose

The purpose of this project is to gain a deeper understanding of software architecture by implementing real-world backend development practices, including:

- Layered architecture with clear separation of concerns
- Dependency Injection
- CQRS with MediatR
- Entity Framework Core
- Object Mapping with AutoMapper
- Validation and Middleware
- Unit Testing
- Scalable and maintainable project structure

---

## 📚 Learning Journey
